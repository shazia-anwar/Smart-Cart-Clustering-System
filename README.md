# Smart Cart Clustering System – Customer Segmentation

## Project Overview
This project performs customer segmentation using unsupervised machine learning techniques. It groups customers based on purchasing behavior to identify patterns and improve business decision-making.

---

## Dataset Features
The dataset includes attributes such as:
- Customer ID
- Annual Income
- Spending Score
- Age
- Gender

---

## Technologies and Tools
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- JupyterLab

---

## Machine Learning Techniques
- K-Means Clustering
- Elbow Method
- Silhouette Score

---

## Project Workflow
1. Data Cleaning and Preprocessing  
2. Feature Selection  
3. Applying K-Means Clustering  
4. Determining Optimal Number of Clusters  
5. Cluster Visualization  

---

## Key Outcome
Distinct customer groups were identified, providing insights into purchasing patterns and customer segmentation strategies.
